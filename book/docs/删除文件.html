删除文件

要从 Git 中删除某个文件，就必须要从已跟踪文件清单中删除，确切地说，是从暂存区域删除，然后提交。可以执行下面的命令删除 README.md 文件，并连带从工作区中删除它。下一次提交时，该文件就不再纳入版本管理了。

git rm README.md

查看分支

查看分支

执行下面的命令，可以查看当前所有分支的列表。分支名前的星号代表现在检出的那一个分支（也就是说，当前 HEAD 指针所指向的分支）。这意味着如果在这时候提交，当前检出的分支将会随着新的工作向前移动。

git branch

如果需要查看每一个分支的最后一次提交，可以执行下面的命令。

git branch -v

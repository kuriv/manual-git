推送标签

默认情况下，推送到远程仓库并不会推送标签。在创建完标签后你必须显式地推送标签到远程仓库上。如果你需要推送标签到简写为 develop 的远程仓库上，可以执行下面的命令。

git push develop v1.8.5

如果想要一次性推送很多标签到简写为 develop 的远程仓库上，可以使用 --tags 选项。这将会把所有不在远程仓库服务器上的标签全部推送到那里。

git push develop --tags
